# KM GH Action

## inputs

### app_id
App id of the KM Syncer app.

### app_private_key
Private key generated for the installed KM Syncer App.

### lib_repo_name
Name of the repository with common-lib.


## Example usage

uses: actions/hello-world-docker-action@v2
with:
  who-to-greet: 'Mona the Octocat'
